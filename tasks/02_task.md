# Tarefa 02 — Instalar Axios e @vueuse/core

## Fase
1 — Fundação

## Depende de
Nenhuma

## Descrição
Instalar as dependências runtime restantes: Axios (HTTP client) e @vueuse/core (composables utilitários).

## Passos
1. Rodar `npm install axios @vueuse/core`
2. Verificar que foram adicionados em `dependencies` no `package.json`
3. Confirmar que `npm run build` passa

## Arquivos modificados
- `package.json`
- `package-lock.json`

## Critérios de aceite
- `axios` e `@vueuse/core` listados em `dependencies`
- Build sem erros
